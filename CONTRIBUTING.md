# Contributing to EvoAgentBench

Thank you for your interest in contributing to EvoAgentBench!

## Development Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run tests: `python -m pytest tests/`

## Code Style

- Follow PEP 8 style guidelines
- Use type hints where appropriate
- Add docstrings to all public functions and classes
- Keep functions focused and small

## Testing

- Write unit tests for new features
- Ensure all tests pass before submitting PRs
- Aim for >80% code coverage

## Commit Messages

- Use clear, descriptive commit messages
- Reference issue numbers when applicable
- Follow the format: "Type: Brief description"

## Pull Requests

1. Create a feature branch
2. Make your changes
3. Add tests
4. Update documentation
5. Submit PR with clear description

## Architecture

See the implementation guide (`EvoAgentBench Guide - Manus.md`) for detailed architecture information.
